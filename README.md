# 🚛 Proyecto Atlas

Proyecto Atlas es una plataforma SaaS orientada a empresas de transporte de carga.

Su objetivo es automatizar la lectura de guías de despacho, centralizar la información logística y ayudar a las empresas a tomar mejores decisiones mediante inteligencia artificial.

## Estado actual

🚧 En desarrollo inicial.

## Primer objetivo

Crear un lector inteligente de guías de despacho que permita:

- Subir una guía en foto o PDF.
- Extraer datos clave.
- Crear una ficha de viaje.
- Guardar historial de vehículos, choferes y clientes.

## MVP

La primera versión buscará leer automáticamente:

- Número de guía.
- Número de transporte.
- Fecha.
- Cliente solicitante.
- Obra destino.
- Chofer.
- RUT chofer.
- Patente tracto.
- Patente carro.
- Dirección destino.
- Hora entrada.
- Hora salida.
- Peso.

## Modelo de negocio

Atlas será un SaaS con suscripción mensual según cantidad de vehículos activos por empresa.
